### Responsive Animations
To make the icons responsive and animated:

```html
<style>
  img {
    transition: transform 0.3s ease-in-out;
  }
  img:hover {
    transform: scale(1.2) rotate(15deg);
  }

  @keyframes rotate {
    0% { transform: rotate(0deg); }
    100% { transform: rotate(360deg); }
  }
</style>
```
